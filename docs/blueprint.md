## Core Features:

- Number Input and Validation: Accept a phone number as input and validate its format.
- WhatsApp Server Connection Simulation: Simulate the process of connecting to WhatsApp servers with messages showing connection status and actions like simulating IP, bypassing firewall, and injecting SQL queries. (purely cosmetic; does not actually perform these actions).
- Redirection to Analysis Page: Redirect the user to a second page that simulates analyzing and processing WhatsApp data.
- Video Playback with Button Trigger: Play a video on the analysis page. At 4 minutes and 13 seconds, trigger a pulsating 'ACESSAR MENSAGENS' (Access Messages) button.
- Simulated Investigation Progress: Display a simulated progress bar from 0 to 100 based on button click from video section. This button click also stops the spinner/loader and displays 3 separate suspicious reports of user. A final redirection is triggered when complete. Purely presentational and for user engagement.

## Style Guidelines:

- Primary color: A vibrant green (#00FF00) to indicate access and activity.
- Background color: A very light, desaturated green (#F0FFF0) to give a high-tech appearance and ensure readability.
- Accent color: A brighter lime green (#32CD32) to create emphasis.
- Body and headline font: 'Inter', a sans-serif font known for its neutral and modern appearance.
- Use a clean, single-column layout with clear visual hierarchy to guide the user through the process.
- Use subtle animations for loading indicators and progress updates to keep the user engaged.
